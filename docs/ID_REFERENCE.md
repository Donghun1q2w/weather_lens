### 지역 코드 구조

지역 코드는 10자리 숫자로 구성:
- **앞 2자리**: 시도 코드
- **3~5자리**: 시군구 코드
- **6~10자리**: 읍면동 코드

예시:
- `1111000001` = 서울특별시 종로구 청운효자동
- `4211000001` = 강원특별자치도 춘천시

## API Endpoints

### Themes
- `GET /api/v1/themes` - 전체 테마 목록
- `GET /api/v1/themes/{theme_id}` - 특정 테마 상세

### Regions
- `GET /api/v1/regions` - 전체 지역 목록 (페이징)
- `GET /api/v1/regions/{region_code}` - 특정 지역 상세
- `GET /api/v1/regions/stats` - 지역 통계

### Marine Zones
- `GET /api/v1/marine/zones` - 전체 해상구역 목록
- `GET /api/v1/marine/zones/{zone_code}` - 특정 해상구역 상세
- `GET /api/v1/marine/{region_code}/forecast` - 지역별 해상예보
- `GET /api/v1/marine/conditions` - 현재 해양 상태
